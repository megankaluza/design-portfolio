Exodus

Exodus | is a cryptocurrency computer app, that is used as a digital wallet to hold Bitcoin and other forms of crypto. Redesigned using Sketch, to display the elements like the periodic table.

Using UX/UI methods, Exodus showcases elements designed to create a more user friendly experience, a case study showcasing the process throughout the process, and a live prototype using inVision.

Created as a sample project for a UX/UI job for Exodus.
