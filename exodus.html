Exodus

Exodus | is a crypto app, that is used as a digital wallet to hold Bitcoin and other forms of online currency. Redesigned, using Sketch and inVision, to display the elements like the periodic table.

Using UX/UI methodology, Exodus showcases elements designed to create a better user friendly experience, a case study explaining the process, and a live prototype using inVision.

Created as a sample project for a UX/UI project, for a company called Exodus.
