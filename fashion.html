Aesthetic

Aesthetic | is a fashion website that outlines the minimum viable product (MVP), and showcases the developmental roadmap throughout the process. Designed in Sketch and hard coded with HTML, CSS, SASS, and jQuery.

Using UX/UI methods, Aesthetic provides the entire process from start to finish, from the design process to front end development. It showcases rough drafts, color choices, typography, mock-ups, frameworks, plugins, responsiveness and API installation.

Created as a final project for my UX/UI Design course at Epicodus, a programming bootcamp.
